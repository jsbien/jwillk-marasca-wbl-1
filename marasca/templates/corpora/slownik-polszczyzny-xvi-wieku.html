The search engine uses the electronic versions of the 33 volumes (containing entries from A to Przypytywać się) of the dictionary of the 16th century Polish, made succesively available since 2008 in the DjVu format in the Kujawsko-Pomorska Digital Library under the terms of the Creative Commons license version Attribution-Noncommercial-No Derivative Works 2.5 Poland.

{% endblocktrans %} {% blocktrans %}

Scanning the first 32 volumes, and later OCR with LizardTech DocumentExpress Professional 5.0, has been done by the library staff.

Volume XXXIII is digitally born, it has been converted to DjVu format by Jakub Wilk using the pdf2djvu and minidjvu programs.

On July 14, 2010 the digitally born versions of volumes XXXI and XXXII have replaced the scans; they have been converted to DjVu format by Jakub Wilk using the pdf2djvu and minidjvu programs.

{% endblocktrans %} {% blocktrans %}

Unfortunately using the digitally-born versions of some other volumes is not possible for formal reasons described in Janusz S. Bień's paper.

{% endblocktrans %} {% blocktrans %}

The OCR results for the scanned volumes and the hidden text layer of volumes XXXI-XXXIII have been converted to the suitable corpus format by Jakub Wilk. The corpus consists of ca. 31 million segments. This is the version 4 of the corpus, available since July 14, 2010.
